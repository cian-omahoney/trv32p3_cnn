Q-2019.12
=============
Reference Methodology in Lynx-RTM version Q-2019.12 provides the 
following new features, enhancements, and changes.

Updated Files
-------------
*  Updated setup script to turn off Metrics and ARO by default
*  Updated ICC2*xml file to match with the latest ICC2 Makefile

P-2019.03-SP4
=============
Reference Methodology in Lynx-RTM version P-2019.03-SP4 provides the 
following new features, enhancements, and changes.

Updated Files
-------------
*  Remove checking for RM version.  All RM tar file that contains 
   rm_in_lynx/ should support running RM in Lynx.
*  Updated DT for app_options comparison in ICC2 tasks
*  Updated view*.tcl script to support "Edit Src"

P-2019.03-SP2

P-2019.03
=========
Reference Methodology in Lynx-RTM version P-2019.03 provides the 
following new features, enhancements, and changes.

Updated Files
-------------
*  Updated the setup script for more robust behavior
*  DC task are now default to use dcnxt_shell 
*  Updated the run_rm.tcl script.
   Create link for Design Compiler and IC Compiler II 
   Reference Methodology (RM) reports in Design Tracker (DT)
*  Use the latest (lynx/2019.03) DT shell script for Design Tracker, including 
   the comparison reports and the Design Compiler Design Summary report.
